REST API endpoint

https://quran-api-7tom.onrender.com/quran/chapter_index/verse_index

The URL follows a RESTful API structure and represents an endpoint for retrieving
a specific verse from the Quran.

https://quran-api-7tom.onrender.com → Base URL (API host)

/quran → Main route indicating Quran-related endpoints

/chapter_index → Placeholder for the chapter (Surah) number

/verse_index → Placeholder for the verse (Ayah) number

Example Usage:

https://quran-api-7tom.onrender.com/quran/1/1 → Would fetch Surah 1, Ayah 1 ("Al-Fatiha:1")

https://quran-api-7tom.onrender.com/quran/2/0 → Would fetch full Surah 2 ("Ayat-ul-Kursi")

Note: This API operates on a cold start mechanism, meaning the first request
after a period of inactivity may have a slower response time.
